# Data Dictionary - Data Immersion & Wrangling

## Dataset Overview
This dataset contains sales transaction data used for analysis and business insights.

| Column Name | Data Type | Description | Business Use |
|-------------|-----------|-------------|--------------|
| Order_ID | String | Unique ID for each order | Track transactions |
| Order_Date | DateTime | Date of purchase | Time-based analysis |
| Customer_ID | String | Unique customer ID | Identify customers |
| Customer_Name | String | Name of customer | Reference |
| Age | Float | Customer age | Demographic analysis |
| Gender | String | Male/Female | Customer segmentation |
| City | String | Customer location | Regional analysis |
| Product | String | Product purchased | Product performance |
| Category | String | Product category | Category analysis |
| Quantity | Integer | Number of items bought | Sales volume |
| Unit_Price | Float | Price per unit | Revenue calculation |
| Total_Sales | Float | Total order value | Revenue tracking |
| Year | Integer | Extracted year | Trend analysis |
| Month | Integer | Extracted month | Monthly trends |
| Month_Name | String | Month name | Reporting |
| Quarter | Integer | Quarter of year | Business reporting |
| Day | Integer | Day of month | Daily analysis |
| Day_Name | String | Weekday name | Sales pattern analysis |
